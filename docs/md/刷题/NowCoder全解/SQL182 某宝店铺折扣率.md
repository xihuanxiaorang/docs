## 答案

```sql
SELECT ROUND(SUM(sales_price) / SUM(st.sales_num * pt.tag_price) * 100, 2) AS `discount_rate(%)`
FROM sales_tb st
         INNER JOIN product_tb pt USING (item_id);
```
